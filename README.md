# Epicodus Student Info

#### Collects and Organizes information of Epicodus students. 11/8/17

#### By Kevin Babcock, Malgorzata Haniszewska, Shane Robinson, Qianqian Hu

## Description

This application will ask for information from Epicodus students, and place the information in the relevant place on the page. The tabs on the right allow the user to choose which list they would like to look at, including the complete student list. If any individual student is clicked on, all of their information will be displayed.

Specs:

* Student name will display below form after submission
* When student's name is clicked from any list, all the information they entered will appear to the right.
* If a language is provided without confidence, it will tell user that there was no confidence provided.
* If a language form is left empty, nothing will be added to student object.
* When any of the links on the left are clicked, all students who match that criteria will display in the list below the form.
* When a language link is selected in the left toolbar, the students confidence in that language will appear next to their name in the list that appears.

## Setup/Installation Requirements

* Go to github.com/yetka
* Search for 'intro-team-project' in the repositories
* Fill out the form to add a student and begin searching

## Known Bugs

Data is not saved anywhere. Everything is lost on page refresh.
